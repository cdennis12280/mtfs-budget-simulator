# Dry‑Run Checklist (Pre‑Pitch)

> Quick links: `docs/QUICK_START.md` for setup/run, `docs/DEPLOYMENT.md` for hosting, `docs/AUTHENTICATION.md` for access control.


## Setup
- Enable Demo Mode in Settings.
- Confirm Council name + reserves policy.
- Confirm role = Admin.

## Dashboard
- Verify S151 Summary values update with assumptions.
- Validate Scenario Story (Base vs Pessimistic).

## Sensitivity
- Run analysis and confirm tornado chart renders.

## Risk Advisor
- Confirm risk register loads.
- Generate stress table and composite scenario.

## Reports
- Generate Executive Summary.
- Generate Board Pack.
- Verify PDF downloads and content.

## Audit
- Confirm export actions are logged.
- Export audit log CSV.

## Demo Reset
- Click “Reset demo tenant”.
- Toggle Demo Mode off/on and confirm baseline resets.
